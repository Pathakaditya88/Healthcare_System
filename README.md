# Full-Stack Healthcare System (MERN)

A secure healthcare management system built with MongoDB, Express, React, and Node.js.

## Features
- **Authentication:** Secure JWT-based login and registration.
- **Health Metrics:** Patients can record vitals (BP, Heart Rate) with automatic critical alert logic.
- **Role-based Access:** Different views for Doctors and Patients.

## Tech Stack
- **Backend:** Node.js, Express, MongoDB Atlas, Mongoose.
- **Frontend:** React, Vite, Tailwind CSS (Coming Soon).
